# Project Summary
The project is a fullstack inventory management system designed to efficiently manage products, medications, and transfers between primary and secondary storages. Built with React and TypeScript for the frontend and Express with PostgreSQL for the backend, it offers a modern UI and robust functionality including user authentication and data export features.

# Project Module Description
- **Frontend**: Built with React and Shadcn-UI, it includes:
  - **Login/Registration**: User authentication with JWT.
  - **Dashboard**: Main interface for managing inventories.
  - **Storage Management**: Add, edit, delete, and transfer products between primary and secondary storages.
  - **Medicamentos**: Manage medication inventory with batch tracking.
  - **Transfer History**: View all transfers made between storages.
  
- **Backend**: Developed using Express and TypeScript, it provides:
  - **API Endpoints**: For authentication, storage management, and medication handling.
  - **Database Connection**: Uses Neon PostgreSQL for data storage and management.
  - **Security**: Implements JWT for secure user sessions.

# Technology Stack
- **Frontend**: React, TypeScript, Shadcn-UI, Vite
- **Backend**: Express, TypeScript, PostgreSQL (Neon)
- **Authentication**: JWT (JSON Web Tokens)
- **Styling**: Tailwind CSS

# Usage
1. **Install Dependencies**:
   ```bash
   cd /workspace/shadcn-ui/server
   pnpm install
   cd /workspace/shadcn-ui
   pnpm install
   ```

2. **Set Up Environment Variables**:
   - Copy `.env.example` to `.env` in both `server/` and root directory and configure your database connection string.

3. **Run the Backend**:
   ```bash
   cd server
   pnpm run dev
   ```

4. **Run the Frontend**:
   ```bash
   pnpm run dev
   ```

5. **Create Your First User**:
   - Use the registration page in the frontend or the API to create an initial user.
